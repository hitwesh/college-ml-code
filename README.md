# College ML Code

This folder contains three Generative AI course assignments, each with its own code, dependencies, and run instructions.

## Assignments

### Assignment 1 - FFNN on MNIST
- Implements a feed-forward neural network on the MNIST dataset.
- Run instructions and output notes are in [College-ml-code/assignment-1/README.md](College-ml-code/assignment-1/README.md).

### Assignment 2 - Gradient Descent Comparison
- Compares gradient descent approaches.
- Run instructions and output notes are in [College-ml-code/assignment-2/README.md](College-ml-code/assignment-2/README.md).

### Assignment 3 - CNN Architectures
- Includes multiple CNN implementations (AlexNet, VGG16, ResNet152, MobileNet).
- Dataset details, split script, and run instructions are in [College-ml-code/assignment-3/README.md](College-ml-code/assignment-3/README.md).

## Quick Start (Per Assignment)
1. Create and activate a virtual environment.
2. Install dependencies from the assignment's requirements file.
3. Run the assignment script(s) listed in that assignment README.

## Notes
- Outputs for all assignments are provided in image format as described in each assignment README.
